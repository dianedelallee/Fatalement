Git Stash

When you have made some modifications and want to achieve another tasks, I could be possible to save your modification by using stash.

git stas

If you want to give a name to your stash, you can use the keyword save

git stash save 'name of my stash'
once you have made your other task you may want to use your stash. if you just want to use your last stash 
>git stash pop
If you have several stash, you can list them in order to see which one you wnat to unstash. 
git stash list
for example to unstash the last one. 
git stash pop stash@{0}
Be careful ! if you use the drop keyword, it will clear and delete your stash 
git stash drop stash@{0}
if you have used drop instead of pop, you still have a way to refund your modificaiton. when you use drop, you will see the hash in the terminal. By using the keyword apply, you should be able to apply your stash but it is not recreating it 
git stash apply your hash
if unfortunately, you have close your terminal, you can use the File System COnsistency check to recover your hash. 
git fsck --no-reflogs | awk '/dangling commit/ {print $3}'
You can also filter by date using the until and since arguments 
git show --oneline --since='2018-09-01' --until='2018-10-01' $(git fsck --no-reflogs | awk '/dangling commit/ {print $3}')
